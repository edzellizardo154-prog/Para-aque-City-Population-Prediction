PARAÑAQUE

POPULATION
ABSTRACT

Abstract: This study presents a predictive modeling approach for urban population growth in Parañaque City using Linear Regression and Polynomial Curve Fitting based on the Least Squares Method. Historical population data from 1990 to 2024 were analyzed to evaluate model performance. Results indicate that Polynomial Regression better captures nonlinear growth trends compared to Linear Regression.

METHODOLOGY
🗂️

Data Collection

Population data were collected from historical census records and analyzed to identify growth trends and changes in population behavior over time.

📈

Linear Regression

Linear Regression applies the equation y = mx + b to estimate population growth using a straight-line trend based on historical population values.

📊

Polynomial Regression

Polynomial Regression captures nonlinear growth patterns and curved trends, allowing more accurate prediction of future population changes.

➗

Least Squares Method

Error minimization was performed using the Least Squares Method to reduce the difference between actual population data and predicted model values.
